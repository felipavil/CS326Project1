Search for Ingredient Substitutes

How To Use

Using the search bar, type the name of the ingredient you want a substitute for.

Add Items

If you search for an ingredient and it isn't found, don't worry! You can add a new substitute directly from the search results:

  1. Type the ingredient name into the search bar and click "Search".
  2. If no substitute is found, you'll see a message saying "No substitute found for this ingredient."
  3. Below this message, a new input field will appear where you can type in the substitute for the ingredient.
  4. After typing the substitute, click the "Submit" button to add it to our database.
  5. The next time you search for this ingredient, your newly added substitute will be available!
